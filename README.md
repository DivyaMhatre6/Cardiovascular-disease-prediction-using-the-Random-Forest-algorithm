❤️ Cardiovascular Disease Prediction using Random Forest
📌 Project Overview

This project focuses on predicting the likelihood of cardiovascular diseases using Machine Learning techniques. The model is built using the Random Forest Algorithm, which analyzes patient health data such as age, blood pressure, cholesterol levels, and lifestyle factors to classify whether a person is at risk of heart disease.

The goal of this project is to assist healthcare professionals in early disease detection and support preventive medical decisions through data-driven insights.

🎯 Objectives
Predict cardiovascular disease risk using patient medical data.
Apply Random Forest classification for accurate prediction.
Perform data preprocessing and feature analysis.
Evaluate model performance using standard metrics.
Provide a scalable healthcare prediction system.
🧠 Machine Learning Algorithm

Random Forest Algorithm

Ensemble learning method.
Combines multiple decision trees.
Reduces overfitting.
Provides high accuracy and robustness.
